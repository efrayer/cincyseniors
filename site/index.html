About This Site

A companion to Think Data Analysis, focused on AI chat and conversational interfaces.

Hi, I'm Eric Frayer

Think Dashboards is my dedicated space for AI chat solutions and conversational interfaces. While Think Data Analysis covers data visualization and analytics, this site focuses on the chatbot and AI assistant projects I've been building with OpenAI, Claude, and open-source LLMs.

These projects are hosted on Azure and built with Node.js, leveraging RAG (Retrieval-Augmented Generation) and web search to create useful, interactive AI experiences.

For more background, visit EricFrayer.com or read my technical posts on Medium.
